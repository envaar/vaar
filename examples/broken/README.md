# Broken Example

The given `.env.example` is an example of a file with linting issues on purpose.

## Contents

- `.env.example` is intentionally not safe to copy as-is.
- The sample shows the kind of problems Vaar reports on malformed dotenv files.

## What This Example Shows

- duplicate keys
- invalid key naming
- spaces around keys, delimiters or values
- trailing whitespace
- repeated blank lines
- a final blank line
- incorrect delimiter usage
