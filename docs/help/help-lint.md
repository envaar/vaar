# Lint Help Guide

This page contains the help reference for `vaar lint`. Refer to [README](../../README.md) for installation, verification and the shortest possible first-run steps. For the full lint walkthrough and rule catalog, see [docs/lint/README.md](../lint/README.md).

## Description of the Command

`vaar lint --help` and `vaar help lint` show the lint help screen. It summarizes the command usage, examples and flags.

To view the lint help, run:

- `vaar lint --help`
- `vaar help lint`

Both commands print the same lint help text.

## Scope

This page only covers the lint help screen. For the command behavior, flag details, output, exit codes and rule reference, use [docs/lint/README.md](../lint/README.md).

## Examples

```bash
vaar lint --help
vaar help lint
```
