# Help Guide

This page contains the top-level help reference for Vaar. Refer to [README](../../README.md) for installation, verification and the shortest possible first-run steps.

## Description of the Command

`vaar --help` and `vaar help` show the root help screen for the CLI. The output is Cobra-generated and lists the public top-level commands and flags that are available in the current release.

To view the root help, run:

- `vaar --help`
- `vaar help`

## What the Root Help Shows

The root help output tells you:

- what Vaar is
- its usage
- available top-level commands
- supported root flags
- where to go next if you want command-specific help

## Scope

This page only covers the root help screen.

Command-specific help pages are linked below:

- [Lint help](./help-lint.md)
- [Completion help](./help-completion.md)

> [!NOTE]
> If you want help for a specific command, use that command's own `--help` flag or its `help` subcommand.
