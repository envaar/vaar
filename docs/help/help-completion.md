# Completion Help Guide

This page contains the help reference for `vaar completion`. Refer to [README](../../README.md) for installation, verification and the shortest possible first-run steps. For the full completion walkthrough, see [docs/completion/README.md](../completion/README.md).

## Description of the Command

`vaar completion --help` and `vaar help completion` show the completion help screen. It summarizes the shell subcommands and the `--no-descriptions` flag.

To view the completion help, run:

- `vaar completion --help`
- `vaar help completion`

Both commands print the same completion help text.

## Scope

This page only covers the completion help screen. For shell setup, install examples and running completions from source, use [docs/completion/README.md](../completion/README.md).

## Examples

```bash
vaar completion --help
vaar help completion
```
